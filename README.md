# 👓 Surtilentes Óptica - Sistema de Gestión de Citas

Sistema web completo para gestión de citas oftalmológicas con integración a Google Sheets.

## 🚀 Deploy en Render

### Pasos Rápidos:

1. **Sube a GitHub** (si aún no lo hiciste)
2. **Ve a [Render.com](https://render.com)** → Crear cuenta gratis
3. **New + → Web Service**
4. **Conecta tu repo:** `MasterCardOficial/Optica`
5. **Configuración:**
   - Name: `optica-surtilentes`
   - Environment: `Node`
   - Build Command: `npm install`
   - Start Command: `npm start`
   - Plan: **Free** ✅
6. **Deploy** → ¡Listo en 5 minutos!

Tu sitio estará en: `https://optica-surtilentes.onrender.com`

## 🛠️ Desarrollo Local

```bash
# Con Node.js
npm install
npm start

# Con Python
python -m http.server 8080
```

## 📧 Configurar Emails

1. Ve a [script.google.com](https://script.google.com)
2. Abre proyecto "API Citas Surtilentes"
3. Ejecuta función `testEndpoint`
4. Autoriza permisos (solo una vez)
